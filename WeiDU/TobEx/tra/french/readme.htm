Introduction

Résumé

TobEx (Throne of Bhaal Extender) est une bibliothèque à édition de liens dynamique (DLL), écrite en C++, qui corrige des bugs du moteur de jeu et étend ce dernier en modifiant l'image virtuelle du BGMain.exe de Throne of Bhaal en mémoire.

TobEx est à la fois une modification pour les joueurs et une ressource pour les créateurs de mods. En bref, les caractéristiques de TobEx sont :


  • Corrections de bug dans de nombreuses parties différentes du moteur de jeu
  • Davantage d'informations de debug
  • Extension de l'interface utilisateur, telle que l'ajout de barres de défilement
  • Altérations optionnelles du comportement du moteur
  • (Modeurs) Externalisation de comportement codé en dur
  • (Modeurs) Déclencheurs (NdT : "triggers"), actions et effets nouveaux et modifiés

Composants

Le premier composant, intitulé "TobEx - Base" est le minimum requis pour utiliser TobEx. Pour les joueurs, ce composant :


  • installe tous les fichiers requis pour utiliser TobEx et modifie l'exécutable du jeu BGMain.exe pour qu'il charge TobEx.dll quand le jeu démarre ;
  • active le type de jeu Arènes en multijoueur ;
  • enregistre les plantages du jeu codés en dur dans TobEx.log et affiche toujours une boîte de dialogue quand un tel plantage se produit ;
  • apporte des corrections au comportement de nombreux effets du jeu, tels que cécité, maladie, régénération et poison ;
  • corrige le compte-rendu de l'expérience acquise ;
  • corrige un bug dans l'attribution des facultés de haut niveau du Clerc-Rôdeur ;
  • corrige un défaut du moteur concernant le ciblage des animations de créature mourante ;
  • corrige un profit indu avec une arme de jet qui n'a pas besoin de munition, causant des dégâts supplémentaires quand on utilise des munitions ;
  • corrige des bugs du moteur avec les ensembles de sons ;
  • optimise le code de recherche dans les sacs pour réduire le ralentissement du jeu lorsqu'il y a trop d'objets à l'intérieur ;
  • augmente la quantité de texte stockée dans la barre de dialogue ;
  • agrandit le parchemin de bulle d'aide ;
  • autorise le défilement des sorts de mage et de prêtre, et de la sélection de kit.

Les composants restants sont des "Extras", décrits ci-après :


  • Appliquer un jet de concentration en cas de dégâts [Travail en cours] : au lieu d'être toujours interrompus lorsqu'ils sont touchés, les jeteurs de sorts ont une chance que leur sort soit interrompu. Ceci est géré par un contrôle de concentration. Un jeteur de sort réussit le contrôle de concentration si (1D20 + chance) > (niveau du sort + dégâts subits). Les détails de ce composant sont susceptibles de changer.
  • Réveiller en cas de dégâts : les créatures sortiront du sommeil quand elles seront touchées.
  • Cécité telle que dans la description du sort : la cécité donne -4 au toucher et 4 points de pénalité d'armure tel qu'indiqué dans la description du sort Cécité dans le jeu.
  • Désactiver la couleur grise de Peau de pierre [C] : supprime le changement de couleur en gris pour un personnage qui a lancé Peau de pierre. Ce composant est identique à tob_hacks, SCS et SCS II.
  • Pas d'interruption de sort pour zéro dégât : le lancement de sort n'est pas interrompu si une créature ne subit pas de dégât (i.e. est immunisée).
  • Utiliser le niveau du jeteur de sorts pour Image miroir [C] : lorsqu'on essaie de dissiper Image miroir, la chance de dissiper est modifiée par le niveau du jeteur de sorts. Ce composant est identique à tob_hacks, SCS, SCS II et Spell Revisions.
  • Permettre le jumelage à toutes les races : autorise toutes les races à se jumeler, au lieu des seuls humains.
  • Permettre de changer d'armure en cours de combat : autorise le changement d'armure au milieu d'un combat.
  • Désactiver le surcroît d'expérience : supprime la fonctionnalité codée en dur qui accorde au groupe 110 % de l'expérience véritablement codée dans les créatures.
  • Désactiver la mise en pause automatique dans l'inventaire : en mode un seul joueur, si le jeu n'est pas en pause avant d'entrer dans l'écran d'inventaire, le jeu ne se mettra plus automatiquement en pause.
  • Désactiver le silence en cas de Charme : ce composant a deux effets ; i) il empêche les créatures de perdre leur doublage sonore quand elles sont charmées, et ii) il permet au joueur d'entamer le dialogue avec les créatures charmées.
  • Restrictions sur les compétences au niveau un : restreint le nombre d'étoiles de compétence que l'on peut donner à une compétence pendant la création d'un personnage de niveau un. Ce composant n'a d'effet que pour les gens qui jouent à Baldur's Gate Trilogy-WeiDU.
  • Rester caché en cas de succès du vol à la tire [C] : si un personnage est invisible ou sous invisibilité améliorée, un vol à la tire réussi n'interrompt pas l'invisibilité. Se cacher dans l'ombre s'interrompra quand même à la fin du round. Ce composant se substitue à tob_hacks s'il est installé.
  • Une attaque lors du repos fait avancer le temps : si le repos est interrompu par des monstres errants, un temps aléatoire pouvant aller jusqu'à un peu moins de huit heures va s'écouler. Toutes les durées de repos d'une heure et plus vont s'afficher dans la barre de dialogue. Les durées de repos inférieures à une heure ne s'afficheront pas dans la barre de dialogue.
  • Sous-titres des dialogues de salutation : Comme dans Baldur's Gate I, ce composant affiche le texte correspondant à la saluation orale dans la barre de dialogue au début du dialogue.
  • Activer les sons d'attaque des animations : Ce composant active les sons d'attaque spécifiques des animations, ce qui rend les combats un peu plus polyphoniques... mélodieux peut-être, ou cacophoniques.
  • Quatre emplacements d'arme dans l'inventaire pour tous : donne à toutes les classes jouables quatre emplacements d'arme dans l'inventaire, comme pour les guerriers. Cela change uniquement l'affichage de l'inventaire mais pas la barre rapide en bas de l'écran.
  • Sous-titres pour les ensembles de sons d'origine : ajoute les sous-titres pour les ensembles de sons pour homme et femme livrés d'origine avec Baldur's Gate II - Throne of Bhaal.
  • Supprimer toutes les restrictions de classe pour les races : autorise toutes les classes à toutes les races, à l'exception des paladins nains puisque les nains ne respectent pas le prérequis de charisme pour devenir paladins.
  • Faire tomber l'inventaire au sol en cas de désintegration / mort due au froid / pétrification : en trois composants distincts ; quand une créature meurt par désintégration, froid ou pétrification fracassants, tout l'inventaire tombe au sol exactement comme pour une mort normale.
  • Activer la pause automatique sur tous les écrans : en mode un seul joueur, la pause automatique fonctionnera lorsque le joueur est sur n'importe quel écran. Auparavant, la pause automatique fonctionnait en mode un seul joueur uniquement quand le joueur était en train de visualiser l'écran principal du jeu.
  • Restreindre les animations d'attaque aux véritables attaques : Retire toutes les attaques 'feintes', où les créatures montraient une animation d'attaque mais ne faisaient pas vraiment de jet de toucher.

Prérequis

TobEx requiert Baldur's Gate II : Throne of Bhaal version 26498. La modification ne fonctionnera pas avec les autres versions de Throne of Bhaal.

TobEx est programmé et testé avant tout sur des systèmes Windows. Il se pourrait qu'il fonctionne sur des systèmes Linux et il est peu probable qu'il fonctionne sur des systèmes Macintosh.

Installation et désinstallation

Installation

  1. Décompressez l'archive TobEx dans le répertoire de votre jeu.
  2. Exécutez Setup-TobEx.exe dans le répertoire d'installation.
  3. Suivez les instructions à l'écran pour installer, réinstaller ou désinstaller des composants.
  4. Installez tous vos autres mods.
  5. Lancez le jeu normalement en exécutant BGMain.exe.
  6. Vous saurez que TobEx tourne en tâche de fond si une fenêtre console supplémentaire apparaît avant que le jeu ne démarre. Si cela ne se produit pas, lancez le jeu en utilisant TobExLoader.exe à la place.

Mise à jour

  1. Exécutez Setup-TobEx.exe dans le répertoire d'installation.
  2. Suivez les instructions à l'écran pour désinstaller tous les composants.
  3. Supprimez le répertoire TobEx dans le répertoire d'installation.
  4. Suivez les instructions d'installation à partir de l'étape 1.

Résolution de problème

Tout d'abord, assurez-vous que TobEx soit vraiment la cause du problème. Vous pouvez énergiquement empêcher TobEx de s'exécuter en renommant le fichier TobEx.dll dans le répertoire de votre jeu.

Pour davantage d'information de debug :


  1. Ouvrez TobEx_ini/TobExCore.ini dans un simple éditeur de texte.
  2. Dans la section [Debug], mettez à 1 toutes les valeurs qui sont à 0.
  3. Essayez de reproduire votre erreur dans le jeu. TobEx pourra parfois signaler quelques informations supplémentaires à propos de l'erreur.
  4. Signalez toute erreur que vous trouvez sur les forums Spellhold Studios. Préparez-vous à ce que l'on vous demande de joindre un .zip du répertoire de votre partie sauvegardée, de WeiDU.log, TobEx.log, TobExCore.ini et TobExTweak.ini.

Compatibilité

Une version redistribuable de TobEx, identique au composant "TobEx - Base" de cette modification, peut être incluse dans d'autres modifications. Si c'est le cas, l'installation détectera automatiquement si une mise à jour est nécessaire et mettra à jour en conséquence. L'installation se poursuivra sans changement si une mise à jour est inutile. Aucune action particulière de l'utilisateur n'est nécessaire.

Il est recommandé que TobEx soit installé en premier dans une installation destinée à de nombreuses modifications.

Il se peut que TobEx ait des problèmes de compatibilité avec des mods qui modifient directement l'exécutable. Veuillez suivre les indications de résolution de problème ci-dessus si vous rencontrez des problèmes et croyez qu'ils puissent être dûs à des problèmes de compatibilité avec d'autres modifications. La liste des modifications qui changent l'exécutable comprend (dans l'ordre alphabétique) :

  • 1PP Extended Palette Entries
  • 1PP Female Dwarves
  • 1PP Thieves Galore
  • A6CharmPatch
  • A6XPPatch
  • Aurora's Shoes and Boots
  • Fishing For Trouble
  • Baldur's Gate Trilogy-WeiDU (BGT-WeiDU)
  • Infinity Animations
  • Sword Coast Stratagems (SCS)
  • Sword Coast Stratagems II (SCSII)
  • Spell Revisions
  • tb#tweaks
  • tob_hacks
  • tob_inject
  • Mod Widescreen
  • Divers patchs de limite d'expérience
